AR Math Quiz – Thesis Documentation

Overview
This repository contains the Final Year Project (FYP) thesis report and supporting images for the project titled “AR Math Quiz".

The project focuses on using Augmented Reality (AR) combined with Machine Learning (ML) concepts to improve mathematics learning for students by making it more interactive and engaging.

Note: This repository does not contain the application source code or APK.  
It is for documentation and academic reference only.

Project Objective
Improve student engagement in mathematics learning
Use AR to visualize math concepts in a real-world environment
Introduce adaptive learning using ML ideas
Reduce dependency on traditional teaching methods

Repository Contents
Thesis Report (PDF) 
Complete FYP documentation including abstract, methodology, system design, and results.
